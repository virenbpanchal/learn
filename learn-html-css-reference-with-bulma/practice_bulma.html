a - Anchor tag

Creates a link to a URL: a web page, a section within a page, an email address... Also called the anchor element, where the a comes from.

href - Defines the target of the link.
absolute URL
#ftr - target an element within the same page.
You can use the mailto protocol. Clicking the link will open the user's email client.

target - Defines in which tab or window the clicked link will show up.
_blank - in a new browsing context - new tab
_self - in current tab
_parent - in the parent browsing context
_top - in the top browsing context

rel - Defines how the link target relates to the current web page.
nofollow - The target is not related. Used for external websites usually.

abbr - Abbreviation tag

Defines an abbreviation, and usually includes its full description
