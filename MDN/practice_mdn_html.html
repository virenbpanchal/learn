Q) What is HTML?
HTML - Hypertext Markup Language
Not a programming language.
It's a markup language.
It is used to "tell the browser how to structure the web pages you visit"
Note :- Tags in HTML are case-insensitive i.e they can be written in uppercase or lowercase. For example, a <title> tag could be written as <title>, <TITLE>, <Title>, <TitLE>, etc. and it will work fine. Best practice, however, is to write all tags in lowercase for consistency, readability and other reasons.

Example -

<p>This is an example sentence.</p>

Here,
1) Opening tag - <p> - it consists name of the element i.e. p here.
2) Closing tag - </p> - same as a opening tag, except that it includes a forward slash before the element name.
3) The content - This is the content of the element, which is in this case just text.
4) The element - The opening tag plus the closing tag plus the content equals the element.
Nesting elements :- Put elements inside other elements. - this is called nesting. Example -

This is an <strong>example</strong> sentence.
